# Presentation

Portfolio personnel / CV en ligne

## Description
Site vitrine présentant mes compétences et mon parcours professionnel.

## Voir le site
Le site est disponible sur GitHub Pages (une fois configuré dans les paramètres du dépôt).

## Technologies
- HTML5
- CSS3
- Design responsive

## Développement local
Pour tester le site localement :
```bash
python3 -m http.server 8080
```
Puis ouvrir http://localhost:8080 dans votre navigateur.
